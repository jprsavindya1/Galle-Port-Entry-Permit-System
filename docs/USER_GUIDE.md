SLPA PERMIT SYSTEM – USER MANUAL (V1.0)
1.	Introduction
The SLPA Port Entry Permit System is designed to digitize and manage the entire permit lifecycle for port entry authorization. The system handles temporary permits, monthly permits, vehicle permits, payment processing, blacklist management, and comprehensive reporting capabilities.

GETTING STARTED
 First Time Login
1.	Go to the system URL provided by your administrator
2.	You'll see the login page

Login Credentials:
1.	Enter your email address
2.	Enter your password
3.	Click "Log in" button
Important Notes:
-	Keep your password secure and don't share it
-	If you forget your password, reset your password via given email
-	If you forget your E-mail address, contact your administrator
-	The system will log you out after 4 hours of inactivity for security
On successful login, the Dashboard will display system statistics and navigation links. Dashboard appearance is different depending on the login role type.

15.	 Creating Permits

1.	Temporary Permit (Daily Entry)
USE THIS FOR: VISITORS, CONTRACTORS, TEMPORARY WORKERS WHO NEED ENTRY FOR 1-29 DAYS.
Step-by-Step Process:

1.	Click "Temporary Permit" on the left menu

2.	Selected Documents Provided:
-	Check "NIC Copy" if NIC provided (29 days)
-	Check "Passport Copy" if passport provided (14 days)
-	Check "Driving License Copy" if license is provided (14 days)

3.	Fill in Personal Information:
-	ID Number: Enter the Identification numbers
-	Full Name: Enter full name in CAPITAL LETTERS
-	Name with Initials: Example: "J.D. SILVA"
-	Designation: Select job title (Manager, Driver, etc.)

4.	Fill in Company Information:
-	Company Name: Select from dropdown or type new name
-	Company Address: Enter company address
-	Residence Address: Enter home address

5.	Check Availability:
-	Click "Check Availability" button

6.	System will verify:
-	Person is not blacklisted
-	Dates are valid and do not overlap with issued permits
-	All required information provided
7.	Fill in Permit Details:
-	Pass Type: Choose "PORT PASS" or other type
-	From Date: Start date of permit
-	To Date: End date (maximum 30 days from start)
-	Issue Type: Free or Paid
-	Reason for Entry: Select purpose (Official Duty, Meeting, etc.)

8.	Add to List:
-	If availability check went through successfully, click "Add to List"
-	Entry is added to your session cart
-	You can add multiple entries before submitting

9.	Review Summary:
-	look at the bottom table  to see all entries
-	You can edit or remove entries if needed

10.	 Submit All:
-	When ready, click "Submit All"
-	System generates payment calculation
-	You'll be redirected to payment invoice

Tips:
-	You can create multiple permits at once (batch creation). After first entry is added to the list, company name and address are saved 
-	Check blacklist before proceeding
-	From date must be today or future date
-	If availability check failed look for the error message showing the reason 
2.	Monthly Permit (30-Day Entry)

USE THIS FOR: PERSONS WHO NEED MONTHLY ACCESS AND HAVE POLICE CLEARANCE.

Step-by-Step Process:

1.	Click "Monthly Permit" on the left menu

2.	Fill in Personal Information:
-	ID Type: Must be NIC (only NIC allowed for monthly permits)
-	NIC Number: Enter 12-digit NIC
-	Full Name: Enter full name in CAPITALS
-	Name with Initials: Example: "A.B.C. PERERA"
-	Designation: Select job title

3.	Fill in Company Information:
-	Company Name: Select from dropdown
-	Company Address: Enter company address
-	Residence Address: Enter home address

4.	Fill in Permit Details:
-	Pass Type: Choose "PORT PASS" or other type
-	From Date: Start date 
-	To Date: End date (automatically selected 30 days from start)
-	Issue Type: Free or paid
-	Reason: Select entry reason

5.	Police Report Information: REQUIRED
-	Police Report Issue Date: When report was issued
-	Police Report Expiry Date: When report expires
-	Check  "NIC Copy Attached"
-	Check  "Police Report Attached"

6.	Check Availability & Submit:
-	Click "Check Availability”
-	Click "Add to List"
-	Review summary and "Submit All"

Tips:
-	Police report is MANDATORY for monthly permits
-	Police report must be valid (not expired)
-	If availability check fails, look for the error message showing the reason 
3.	Vehicle Permit

USE THIS FOR: VEHICLES THAT NEED PORT ENTRY AUTHORIZATION.

Step-by-Step Process:

1.	Click "Vehicle Permit" on the left menu

2.	Fill in Vehicle Information:
-	Vehicle Number: Enter registration (e.g., ABC-1234)
-	Owner NIC: Enter vehicle owner's NIC
-	Vehicle Type: Select from dropdown (Vehicle category list)
-	From Date: Start date
-	To Date: End date

3.	Fill in Owner Information:
-	Owner Name: Full name of vehicle owner
-	Owner Address: Owner's address
-	Company Name: Company the vehicle belongs to

4.	Fill in Permit Details:
-	Issue Type: Free or Paid
-	Reason for Entry: Select purpose
-	Remarks: Any additional notes (optional)

5.	Vehicle Documents:
-	Revenue License Number: Enter license number
-	Insurance Number: Enter insurance policy number
-	Check  "Revenue License Copy"
-	Check  "Insurance Copy"

6.	Check Availability & Submit:
-	Click "Check Availability"
-	System checks if vehicle is blacklisted
-	Click "Add to List"
-	Review summary and "Submit All"

Tips:
-	Revenue license and insurance must be valid
-	One permit per vehicle
-	Check if vehicle is blacklisted before processing
16.	Payment & Invoices
AFTER SUBMITTING PERMITS BATCH, YOU WILL REDIRECT TO PAYMENT SUMMERY PAGE.

UNDERSTANDING PAYMENT CALCULATION
The system automatically calculates payment based on:

1.	1.Permit Type:
-	Temporary Permit: LKR 30.00 per day  
-	Monthly Permit: LKR 500.00 per permit
-	Vehicle Permit: Based on vehicle type (LKR 50.00 – 8500.00) per permit

2.	Additional Charges:
-	SSL: Social Security Contribution Levy: 2.5% of base amount
-	VAT: As per current rates (18%)
After receiving payment from the person press confirm Payment
VIEWING INVOICE
After receiving payment from the person press confirm Payment
1.	System automatically redirects to Invoice Page
2.	You'll see:
-	Submission ID – Group identification number for this batch
-	Invoice ID - Payment reference number
-	List of all permits in this batch
-	Individual charges for each permit
-	Total amount to be paid

3. System Records:

22.	Blacklist Management
ADMIN ACCESS REQUIRED

What is Blacklist?
The blacklist prevents certain people or vehicles from getting permits. Used for:
-	Security concerns
-	Previous violations
-	Restricted access
-	Banned individuals/vehicles


1.	Adding to Blacklist

1.	Click "Blacklist Management"** (in Admin menu)
2.	Click "Add to Blacklist" button
3.	Fill in information:
-	NIC/Vehicle Number: Enter ID or registration
-	Name: Person or vehicle owner name
-	Reason: Why being blacklisted (required)
4.	Click "Add to Blacklist"

What happens:
-	Entry added to blacklist immediately
-	System will block permit creation for this person/vehicle
-	Blacklist history recorded

2.	Viewing Blacklist

1.	Go to "Blacklist Management"
3.	You'll see table with:
-	NIC/Vehicle Number
-	Name
-	Reason for blacklisting
-	Date added
-	Added by (username)
-	Actions
3.	Removing from Blacklist

1. Find the entry on the blacklist table
2. Click  Remove button
3. Confirm removal
4. Entry removed immediately

Note: Removal is recorded in blacklist history for audit purposes.

4.	Blacklist History

Purpose: Track all blacklist changes for security and accountability.
1. Click "View History" button
2. See complete log:
-	What was added/removed
-	When it happened
-	Who did it
-	Reason provided

5.	Exporting Blacklist

For reports or external use:
1. Excel Export:
-	Click "Export Excel" button
-	File downloads automatically
-	Open in Microsoft Excel

2. PDF Export:
-	Click "Export PDF" button
-	File downloads automatically
-	Open in PDF reader

 Use cases:
-	Share with security team
-	Monthly reports
-	Backup records

6.	How Blacklist Checking Works

Automatic Checking:
7.	When creating any permit, the system automatically:
1.	Checks entered NIC/vehicle number
2.	Searches blacklist database in Availability check button
3.	If found:
-	Blocks permit creation
-	Shows error message: "Person/Vehicle is blacklisted"
-	Displays blacklist reason
4.	If not found:
-	Allows permit creation to continue

YOU DON'T NEED TO MANUALLY CHECK - SYSTEM DOES IT AUTOMATICALLY WITH AVAILABILITY CHECK BUTTON!

26.	Troubleshooting

Common Issues and Solutions

PROBLEM: "SESSION HAS EXPIRED" MESSAGE

Cause: You've been inactive for 4 hours.

Solution:
1.	Click "OK" on the message
2.	Log in again
3.	Your work is saved if you submit it

Prevention:
-	Click around the system every hour to stay active
-	Save/submit work regularly
-	Don't leave browser idle too long

PROBLEM: "PERSON/VEHICLE IS BLACKLISTED"

Cause: The NIC or vehicle number is in the blacklist.

Solution:
1.	Verify the NIC/vehicle number is correct
2.	Check with administrator
3.	If legitimate, admin must remove from blacklist first
4.	Then retry permit creation



PROBLEM: CANNOT FIND A PERMIT

Cause: Wrong search criteria or permit doesn't exist.

Solution:
1.	Double-check search terms:
-	Verify spelling
-	Check NIC number is correct
-	Try to permit ID if you have it
2.	Check if permit is cancelled
3.	Try broader search (partial name)

PROBLEM: PRINT BUTTON NOT WORKING

Cause: Browser popup blocker or printer not ready/ power failed .

Solution:
1.	reprint permits from dashboard permit view link
1.	Find last invoice number
2.	reprint batch again
2.	Check printer:
-	Is printer turned on?
-	Is printer connected?
-	Try printing a test page
3.	Contact IT support if still not working






PROBLEM: "INVALID DATE RANGE" ERROR

Cause: Dates are not logical or out of range.

Solution:
1.	Check From Date is before To Date
2.	Check From Date is today or future (not past)
3.	Check date range doesn't exceed maximum:
-	Temporary: 29 days maximum
-	Monthly: Exactly 30 days
-	Vehicle: As per policy (daily 28 days maximum, monthly 29 days, annually 364 days)

PROBLEM: WRONG PAYMENT AMOUNT SHOWING
Cause: Incorrect vehicle type or rate settings.

Solution:
1.	Check vehicle type selected correctly
2.	Verify rates in Payment Settings (admin)
3.	If wrong, cancel and recreate permit with correct info
4.	Contact administrator to verify payment rates

PERMIT ID FORMAT

UNDERSTANDING PERMIT IDS:

- **TP251201001** = Temporary Permit, Year 25, Month 12, Number 0001
- **MP251201002** = Monthly Permit, Year 25, Month 12, Number 0002  
- **VH251201003** = Vehicle Permit, Year 25, Month 12, Number 0003

DOCUMENT REQUIREMENTS CHECKLIST

For Temporary Permits:
-	NIC copy OR Passport copy OR Driving License copy
-	Valid ID (not expired)
-	Clear, readable copies

For Monthly Permits:
-	NIC copy (mandatory)
-	Police report (mandatory)
-	Police report must be valid (not expired)

For Vehicle Permits:
-	Revenue license copy (mandatory)
-	Insurance copy (mandatory)
-	Both must be current and valid
-	Owner NIC copy
